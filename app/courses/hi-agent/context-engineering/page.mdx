# M04 · Context Engineering

> Prompt Engineering 处理的是"说什么"，**Context Engineering 处理的是"让模型看到什么"**。

## 本章目标

- 理解 Context 与 Prompt 的本质区别
- 掌握 Context 组装与压缩的常用策略
- 认识上下文窗口、检索增强、长期上下文的工程方案

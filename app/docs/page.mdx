# 课程简介

欢迎来到 **Hi-Agent**——一门关于 AI Agent 工程的系统课程。

这门课不讲玄学，也不讲概念堆砌。我们只关心一件事：**如何把 LLM 真正变成一个能用、好用、可控的 Agent 系统**。

<Callout type="info" emoji="📎">
  本课程以**文字 + 示例代码**为主，每一章都可以独立阅读，但建议按顺序循序渐进。
</Callout>

## 你会学到什么

- **Chat**：从最基础的对话接口出发，理解消息、角色、流式输出与协议层。
- **Agent Loop**：搞清楚 Agent 的核心循环是如何运转的。
- **Tool**：让模型正确地调用工具——这是一切能力扩展的起点。
- **Context Engineering**：Prompt 不是一切，Context 才是。
- **Memory**：让 Agent 真正"记住"用户、记住任务、记住自己。
- **Multi-Agent**：从单体走向群体，理解协作与编排。
- **Harness**：运行时、沙箱、评测——让 Agent 从 Demo 走向生产。

## 适合谁读

- 已经写过一些 LLM 应用，但不清楚"Agent"到底意味着什么
- 想把原型推向生产、却被各种工程问题绊住的开发者
- 希望建立一套完整 Agent 心智模型的技术人
